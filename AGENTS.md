## 프로젝트 개요

Keycloak 인증/권한 관리 실습 프로젝트입니다. 현재 초기 설정 단계이며, 코드가 추가되면 이 파일을 업데이트하세요.

## 예상 기술 스택

Keycloak은 Java 기반 오픈소스 IAM(Identity and Access Management) 솔루션입니다. 이 프로젝트에서 사용할 가능성이 높은 기술:

- **Keycloak**: 인증 서버 (Docker로 실행 권장)
- **Java / Spring Boot**: 클라이언트 애플리케이션 (예상)
- **Docker / Docker Compose**: 로컬 개발 환경
