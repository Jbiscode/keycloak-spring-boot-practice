# Project Overview

## Purpose
Keycloak 인증/권한 관리 실습 프로젝트 (keycloak_v1)
IAM(Identity and Access Management) 학습 및 실습을 위한 프로젝트로, 현재 초기 설정 단계.

## Tech Stack (예상)
- **Keycloak**: 인증 서버 (Docker로 실행 권장)
- **Java / Spring Boot**: 클라이언트 애플리케이션
- **Docker / Docker Compose**: 로컬 개발 환경

## Status
초기 단계 - 아직 실제 코드 없음. 파일 추가 후 이 메모리를 업데이트할 것.
