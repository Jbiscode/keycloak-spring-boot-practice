# Suggested Commands

## System (Darwin/macOS)
- `ls`, `find`, `grep`, `cat` — 파일 탐색
- `git status`, `git log`, `git diff` — git 작업
- `docker ps`, `docker compose up -d` — Docker 환경 관리

## Keycloak (Docker)
- `docker compose up -d` — Keycloak 서버 실행 (설정 후)
- `docker compose down` — 서버 중지

## 코드 추가 후 업데이트 필요
- 빌드 명령어 (Maven: `mvn clean install` / Gradle: `./gradlew build`)
- 테스트 실행 명령어
- 로컬 실행 명령어
